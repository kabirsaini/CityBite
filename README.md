CityBite – Restaurant Management and Food Ordering site 
Technology Used - MongoDB, ExpressJs, ReactJs, NodeJs (MERN stack), React Recharts, Razorpay Payment Gateway, Google oAuth.
◦ Designed and developed a multi-user (user/vendor) food ordering and restaurant management platform 
  using the MERN stack and RESTful APIs.
◦ Enabled Google OAuth with JWT authentication for secure and simplified login/registration.
◦ Integrated Razorpay payment gateway to handle secure online payments.
◦ Built separate dashboards for restaurant owners and users with real-time order management.
◦ Developed interactive charts and graphs using Recharts to help vendors analyse monthly visits and orders.
